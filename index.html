Advanced Examples of Fourier and Laplace Transforms

Fourier Transform Examples

Example 1: Fourier Transform of a Rectangular Pulse

Given a rectangular pulse defined as:

f(t) = 1 for |t| < T/2

Step-by-step solution:

  1. Use the definition of the Fourier transform: F(ω) = ∫ f(t) e-jωt dt
  2. Integrate over the pulse width: F(ω) = ∫-T/2T/2 e-jωt dt
  3. Solve the integral: F(ω) = (2/ω) sin(ωT/2)

Example 2: Fourier Transform of a Sine Wave

Given a sine wave f(t) = sin(ω0t)

Step-by-step solution:

  1. Use Euler's formula: sin(ω0t) = (ejω0t - e-jω0t) / 2j
  2. Apply the Fourier transform: F(ω) = π[δ(ω - ω0) - δ(ω + ω0)] / 2j

Example 3: Fourier Transform of an Exponential Decay

Given f(t) = e-at for t ≥ 0

Step-by-step solution:

  1. Set up the integral: F(ω) = ∫0∞ e-(a+jω)t dt
  2. Solve the integral: F(ω) = 1 / (a + jω)

Example 4: Fourier Transform of a Gaussian Function

Given f(t) = e-at2

Step-by-step solution:

  1. Use the known result for Gaussian functions: F(ω) = √(π/a) e-ω2 / 4a

Example 5: Fourier Transform of a Delta Function

Given f(t) = δ(t)

Step-by-step solution:

  1. Apply the Fourier transform definition: F(ω) = ∫ δ(t) e-jωt dt
  2. Result is: F(ω) = 1

Laplace Transform Examples

Example 1: Laplace Transform of f(t) = e-at

Step-by-step solution:

  1. Set up the integral: L{e-at} = ∫0∞ e-(a+s)t dt
  2. Solve the integral: L{e-at} = 1 / (s + a)

Example 2: Laplace Transform of f(t) = t

Step-by-step solution:

  1. Set up the integral: L{t} = ∫0∞ t e-st dt
  2. Solve the integral using integration by parts: L{t} = 1 / s2

Example 3: Laplace Transform of f(t) = sin(bt)

Step-by-step solution:

  1. Set up the integral: L{sin(bt)} = ∫0∞ sin(bt) e-st dt
  2. Use the known result: L{sin(bt)} = b / (s2 + b2)

Example 4: Laplace Transform of f(t) = cos(bt)

Step-by-step solution:

  1. Set up the integral: L{cos(bt)} = ∫0∞ cos(bt) e-st dt
  2. Use the known result: L{cos(bt)} = s / (s2 + b2)

Example 5: Laplace Transform of f(t) = t e-at

Step-by-step solution:

  1. Set up the integral: L{t e-at} = ∫0∞ t e-(a+s)t dt
  2. Solve using integration by parts: L{t e-at} = 1 / (s + a)2

Conclusion

These examples illustrate the application of Fourier and Laplace transforms to various functions. Both transforms are integral tools in engineering and applied mathematics, useful for analyzing and solving complex problems in the time and frequency domains.
